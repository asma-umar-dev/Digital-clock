# ⏱ Digital Clock

**Live Site:** [asma-umar-dev.github.io/Digital-clock](https://asma-umar-dev.github.io/Digital-clock/)

A clean, minimal live digital clock web app that displays the current time in real time, along with the date and a visual seconds progress indicator.

## ✨ Features

- **Live Clock** — Real-time display of Hours : Minutes : Seconds
- **AM/PM Indicator** — 12-hour format with AM/PM label
- **Date Display** — Shows the current date
- **Seconds Progress Bar** — Visual progress indicator that tracks the current second within the minute
- **Minimal, Responsive UI** — Clean single-screen design, works on desktop and mobile

## 🛠️ Tech Stack

- HTML5, CSS3, JavaScript (vanilla, single-page static app)
- Uses native JavaScript `Date` object, updated every second
- Hosted via **GitHub Pages**

## 📁 Getting Started

Clone the repo and open `index.html` directly, or serve it locally:

```bash
git clone https://github.com/asma-umar-dev/Digital-clock.git
cd Digital-clock
# then open index.html in your browser
# or serve locally:
npx serve .
```
